# Windows AD Labs

This section documents my Windows Server and Active Directory labs.

## Labs

- **Lab 01 – Standalone Domain Controller**
  - Install Windows Server
  - Promote to Domain Controller
  - Configure AD DS + DNS
  - Create test users and groups

- **Lab 02 – DNS & DHCP Integration**
  - Install and configure DHCP
  - Scope options for clients
  - Verify DNS registration

- **Lab 03 – OU & GPO Design**
  - Create OU structure (Users, Computers, Servers)
  - Apply baseline GPOs
  - Test with a joined client

## Notes

- Server version: Windows Server 2022 Datacenter (Eval)
- Hypervisor: VirtualBox (running on Windows 10/11)
- Networking: Host-only / NAT combo for lab isolation
